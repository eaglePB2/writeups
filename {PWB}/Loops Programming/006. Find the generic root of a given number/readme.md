Generic root is the mathematical formula to calculate the sum of all digits in a given number until we get a single-digit output (less than 10).

### Sample 1
`Input`
```
98765
```

`Output`
```
8
```

`Explanation`
```
98765 = 9 + 8 + 7 + 6 + 5 = 35 -> 3 + 5 = 8 (done)
```

### Sample 2
`Input`
```
101325
```

`Output`
```
3
```
