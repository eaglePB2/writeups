Write a program to calculate two complex numbers by sum and minus.  
User inputs 2 complex numbers.  
The Output should be the sum and the difference of two complex numbers.  

### Sample 1
`Input`
```
3.5 6.5
45.2 69.96
```

`Output`
```
48.7+76.5i
-41.7-63.5i
```

### Sample 2
`Input`
```
12 2
-10 -3
```

`Output`
```
2.0 - 1.0i
22.0 + 5.0i
```
