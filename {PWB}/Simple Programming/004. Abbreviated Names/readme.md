Abbreviated is a short-form name, given example in Samples Sections.

Write a Program to Convert a person's name in Abbreviated.

### Sample 1
`Input`
```
Eric Unreal Gurt
```

`Output`
```
E.U.G
```

### Sample 2
`Input`
```
Kosmita bin Ahmad
```

`Output`
```
K.b.A
```

### Sample 3
`Input`
```
Lim Kee Siang
```

`Output`
```
L.K.S
```
