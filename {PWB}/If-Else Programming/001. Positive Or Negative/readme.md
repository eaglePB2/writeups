Write A C Program To Check Number Is Positive Or Negative.  
Number should be Entered Through User Then program Print the Output.  

If the user input is 0, print "Neutral" instead.

### Sample 1
`Input`
```
6
```

`Output`
```
Positive
```

### Sample 2
`Input`
```
-19
```

`Output`
```
Negative
```

### Sample 3
`Input`
```
0
```

`Output`
```
Neutral
```
