Create a Program To Check Character Is:
- Uppercase( Like A,B,C........Z)
- Lowercase ( Like a,b,c.....z )
- Digit ( Like 1,2,3,.......9 )
- Symbol ( Like #,@,<,> )

### Sample 1
`Input`
```
j
```

`Output`
```
Lowercase
```

### Sample 2
`Input`
```
0
```

`Output`
```
Digit
```

### Sample 3
`Input`
```
*
```

`Output`
```
Symbol
```
